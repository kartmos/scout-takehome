# Task prompts

Store task prompts prepared with Codex in this directory.

Use numbered, descriptive names:

- `001-backend-bootstrap.md`
- `002-photo-api.md`
- `003-thumbnail-engine.md`
- `004-gallery.md`

Keep one implementation task in each file. Start from `TEMPLATE.md`, then run it in Claude Code:

`/implement-task @.claude/prompts/001-backend-bootstrap.md`

Prompt files are a curated record of important tasks. Full chat transcripts do not need to be committed.
